# cPanel - Tools

1. Click on WordPress Management

2. Click on Log in

# Error de privacidad

3. Click on Plugins 1

4. Click on Add Plugin

5. Click on Add Plugin

6. Click on Upload Plugin

7. Select "all-in-one-wp-migration.7.94.zip" from file upload menu

8. Click on Install Now

9. Click on Activate Plugin

10. Click on Add Plugin

11. Click on Upload Plugin

12. Select "all-in-one-wp-migration-unlimited-extension (1).zip" from file upload menu

13. Click on Install Now

14. Click on Main menu

15. Click on Activate Plugin

# cPanel -

16. Click on cPanel logo

17. Click on MultiPHP INI Editor

18. Select conexiones.meritop.com

19. Click on txtmemory_limit

20. Click on memory_limit This sets the maximum amount of memory in bytes that a script is allowed to allocate. This helps prevent poorly written scripts for eating up all available memory on a server. Note that to have no memory limit, set this directive to -1.

21. Type "2048M"

22. Click on Apply

23. Click on cPanel logo

24. Click on File Manager

25. Click on public_html

26. Click on wp-content…

27. Click on ai1wm-backups…

28. Click on Upload

29. Click on Select File

30. Select a file from upload menu

31. Click on Go Back to “/home/wpconexprod/public_html/wp-content/ai1wm-backups”

# Plugins ‹ Calculating Infinity — WordPress

32. Click on Backups

33. Click on More

34. Click on Restore

35. Click on PROCEED >

36. Click on FINISH >

37. Paste "carlosperez" into input

38. Paste input

39. Click on Acceder

40. Right click on Visitar el sitio
